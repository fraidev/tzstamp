TzStamp Web

Repository TzStamp CLI

Prove possession of a file at a certain time.

Instructions

1. Click the file submission button to choose a file, it is then (locally, in your browser) SHA-256 hashed. Alternatively, hash it yourself and paste the hex digest into the form.

2. Press the 'stamp' button and a link(s) appears in the textbox below.

3. Open the link(s) and wait for the next merkle root to be published (see schedule), then refresh.

4. Right click and save the page. You now have proof that your hash was included in the block.

5. Store multiple copies of this file in safe places (consider printing it out). Without it you can't verify your timestamp!
