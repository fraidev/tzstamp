TzStamp API

Stamp A File

POST /api/stamp/

Adds a hash to the next published merkle root.

JSON Params

Name Type Description
hash string SHA-256 hash to be included in the next merkle root.

Example

{
"hash":
        "40cb6ac0645f648a12d0aab7df480796fc4165609a65ea430018d9b3af68b76d"
}

Retrieve A Proof

GET /api/proof/{id}

Get proof associated with hash {id}. Returns:

200 if proof available.

202 if proof waiting to be published.

404 if proof no longer in server cache.
